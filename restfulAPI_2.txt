restframework 만들때 순서
1. model 설계
2. model -> serializer (이름, 전화, 주소 등을 필드로 만듦.)
3. url [~/addresses] -> 주소록 model을 가져오는 url
*일반적*
MVC 패턴 
Model : 데이터 형식
View : 보여주는 곳
Control. : 데이터를 가공하는 곳
*장고*
Model : 데이터 형식
View : 데이터 가공하는 곳
Template : 보여주는곳
