restful API 
* url을 어떻게 쓸것인가 -> url 규칙이있다.
  http://~/member/1
  http://~/club
* CRUD 호출할때 방식을 하나로 통일하자. 
  Create : 생성(POST)
  Read : 조회(GET)
  Update : 수정(PUT)
  Delete : 삭제(DELETE)
    ~/member/1 GET
    ~/member/1 PUT

개발환경: 파이참, python 3.8

가상환경을 왜 쓰는지?
개발을 하다보면 프로젝트 별로 쓰는 라이브러리들의 각기 다른 버전들을 사용해야할
경우가 생긴다.
